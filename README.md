# Student Record Management System (C Program)

A **console-based Student Management System** written in C that allows users to manage student records efficiently.  
The program supports adding, searching, modifying, deleting, and saving student information to a file.

---

## Features

- Display all students
- Add a new student
- Search for a student by roll number
- Modify student details
- Remove a student
- Remove all student records
- Load student records from a file
- Save student records to a file
- Calculate and display average marks (if implemented)
- Simple welcome message and main menu on program start

---

## Prerequisites

- GCC compiler (MinGW for Windows or GCC for Linux/Mac)
- PowerShell or any terminal (for Windows users, run `chcp 65001` to properly display emojis if used)
